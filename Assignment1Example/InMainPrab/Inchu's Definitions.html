Inchu's HTML Definitions

Definitions table


Tag Definition
!DOCTYPE html This tag can is used to tell the web browser which HTML version the page uses
html This tag tells the browser that it is an HTML document
body This tag defines the main body of the document. It includs all the contents of the document (such as text, hyperlinks, images, tables, lists, etc.)
table This tag defines an HTML table. we can use it to create a table in HTML
tr This tag defines a row in an HTML table. we can make a row in an HTML table
th This tag defines a header cell in an HTML table. It often shows on the top of the table
td This tag defines a standard cell in an HTML table. It often use to show data
br This tag can insert a simple line break. It's function is like "Enter key" when use Word App
