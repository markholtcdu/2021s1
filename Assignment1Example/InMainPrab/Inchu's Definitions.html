Definitions table


Tag Definition
!DOCTYPE html This tag is used to tell the web browser which HTML version the page uses
html This tag tells the browser that it is an HTML document
body This tag defines the main body of the document, including all the contents of the document (such as text, hyperlinks, images, tables, lists, etc.)
table This tag defines an HTML table
tr This tag defines a row in an HTML table
th This tag defines a header cell in an HTML table
td This tag defines a standard cell in an HTML table
br This tag can insert a simple line break
