Prabesh's HTML Definitions
HTML codes Definition
!DOCTYPE html The first line of a web page to show it is html.
html A code that is used to structure a web page and its content.
body It defines as documents such as headings, paragraphs, images,etc.
table It used for arranging the data in rows and columns.
tr It define as a row cell.
th A cell as header of a group of table.
td It contains the data in a table.
br It produce a line breaker in text.
a href"..index.html" It creates a hyperlink to another webpage.
img src="images/file.jpg" It will display an image on the page.
ul It represents an unordered list of items.
ol Tt represents an ordered list of numeral or alphabetical.
li It rrepresents an item in an list.
h1 It is the first header visible on a page.
