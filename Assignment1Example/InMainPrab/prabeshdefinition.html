Prabesh's HTML Definitions
HTML codes Definition
!DOCTYPE html This informs the computer that this is a HTML5 document.
html This is the foundation element of an html file.
body This contains all the contents, for example: paragraphs, images, hyperlinks, etc...
table This will create a board on the page.
tr This will define the row of cells in the board.
th This is the header cell of the board.
td This will contain the data of the cells in the board.
br This will create a line break.
a href"..index.html" This will create a hyperlink to another webpage.
img src="images/file.jpg" This will display an image on the page.
ul This represent an unordered list.
ol This represent an ordered list of numeral or alphabetical.
