Mainamala's HTML Definitions

a href="../index.html" Defines a hyperlink The href attribute specifies the URL of the page the link goes to
img src="images/file.jpg" Defines an image
ul Defines an unordered list
ol Defines an ordered list
li Defines a list item
h1...h5 Defines HTML headings
hr Defines a thematic change in the content
br Inserts a single line break
